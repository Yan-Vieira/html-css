Principais Formatações

Negrito / Destaque

Neste parágrafo, temos um termo em negrito usando a tag "<b>" (que é uma tag não semântica)

Neste parágrafo, temos um termo em destaque utilizando a tag "<strong>" (que é uma tag semântica)

Itálico / Ênfase

Neste parágrafo, temos um termo com ênfase utilizando a tag "<i>" (uma tag não semântica)

Neste parágrafo, temos um termo com ênfase utilizando a tag "<em>" (uma tag semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto Grande e Texto Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto Deletado

Podemos marcar um texto como excluído para indicar que o texto deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar um ênfase e indicar que o texto foi adicionado depois.
